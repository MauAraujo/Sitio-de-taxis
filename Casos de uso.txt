Usuarios: Administrador, Conductor, Pasjero.

Casos de Uso:

	 Administrador: Administrar Unidad, Administrar Conductor, Gestionar Tarifa.

	 Conductor: Gestionar Viaje.

	 Pasajero: Solcitar Viaje, Gestionar Viaje, Pagar, Gestionar Credenciales.

Detalles de casos de uso:

	Administrar Unidad:
		- Alta Unidad
		- Baja Temporal
		- Baja Definitiva
		- Actualizar Datos
	Administrar Conductor:
		- Alta Conductor
		- Baja Temporal
		- Baja Definitiva
		- Actualizar Datos
	Gestionar Tarifa:
		- Definir Tarifa Unitaria
		- Modifcar Tarifa Unitaria
	Gestionar Viaje: 
		- Aceptar Viaje
		- Rechazar Viaje
	Solicitar Viaje:
		- Solicitar Viaje
		- Establecer Destino
		- Establecer Origen
	Gestionar Viaje:
		- Modificar Destino
		- Revisar Tarifa
		- Cancelar Viaje
	Pagar:
		- Pagar Viaje
	Gestionar Credenciales:
		- Alta Usuario
		- Modificar Datos Personales
		- Modifcar Datos Pago
		- Baja Usuario
